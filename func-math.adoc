= Math

All math functions are availabke under **Math** package. They can be used by importing 
math package in the script or using package name before the function name. Package name 
together with function name used in documwntation. 

Example with import:

.Example
[source,c]
----
import Math;
println(Atan2(1, 0.5)); //Result: 1.107149
----

Example without import:

.Example
[source,c]
----
println(Math::Atan2(1, 0.5)); //Result: 1.107149
----
