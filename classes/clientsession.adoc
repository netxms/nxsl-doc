Client flags

ID of client session

loginName => String::
Login name

User name

user => User::

userId => Integer::
User ID

[[clientsession-webServerAddress]]
webServerAddress => String::
Address for web server if it is web type connection

[[clientsession-workstation]]
workstation => String::
Workstation information if available
