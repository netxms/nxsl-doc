[.nxsl-class]
[[class-table]]
=== Table

Represents table object (usually it's value of table DCI).

==== Instance attributes

[.attr-list]
--
[[table-columnCount]]
columnCount: int::
Number of columns.

[[table-columns]]
columns: <<class-array,Array>><TableColumn>::
Array of <<class-tablecolumn,column definitions>>.

[[table-instanceColumns]]
instanceColumns: <<class-array,Array>><TableColumn>::
Array of <<class-tablecolumn,column definitions>> including only columns marked as instance columns.

[[table-instanceColumnIndexes]]
instanceColumnIndexes: <<class-array,Array>><Integer>::
Array with indexes of columns marked as instance columns.

[[table-rowCount]]
rowCount: int::
Number of rows.

[[table-rows]]
rows: <<class-array,Array>><TableRow>::
Array of rows with data.

[[table-title]]
title: <<class-string,String>>::
Title of table.
--
