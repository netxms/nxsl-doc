[.nxsl-class]
[[class-deploymentpackage]]
=== DeploymentPackage

Represents a software deployment package with metadata for installation on managed nodes. DeploymentPackage objects are typically returned by <<func-getavailablepackages,GetAvailablePackages()>> function.

==== Instance attributes

[.attr-list]
--
[[deploymentpackage-command]]
command => String::
Installation command to execute.

[[deploymentpackage-description]]
description => String::
Package description.

[[deploymentpackage-fileName]]
fileName => String::
Path to the package file.

[[deploymentpackage-id]]
id => Integer::
Unique package identifier.

[[deploymentpackage-name]]
name => String::
Package name.

[[deploymentpackage-platform]]
platform => String::
Target platform (e.g., "Windows", "Linux").

[[deploymentpackage-type]]
type => String::
Package type identifier.

[[deploymentpackage-version]]
version => String::
Package version string.
--
