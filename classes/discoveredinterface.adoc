[.nxsl-class]
[[class-discoveredinterface]]
=== DiscoveredInterface

Represent discovered interface objects in <<class-discoverednode,discovered node class>>

==== Instance attributes

[.attr-list]
--
[[discoveredinterface-alias]]
alias: <<class-string,String>>::
Interface alias (usually value of SNMP ifAlias)

[[discoveredinterface-chassis]]
chassis: int::
Chassis id

[[discoveredinterface-description]]
description: <<class-string,String>>::
Interface description

[[discoveredinterface-index]]
index: int::
Interface index

[[discoveredinterface-ipAddressList]]
ipAddressList : <<class-array,Array>>::
Array with <<class-inetaddress>> objects, that represent all adresses that this interface has

[[discoveredinterface-isPhysicalPort]]
isPhysicalPort: bool::
`TRUE` if this interface object represents physical port

[[discoveredinterface-macAddr]]
macAddr: <<class-string,String>>::
String representation of MAC address separated by ":"

[[discoveredinterface-module]]
module: int::
Module

[[discoveredinterface-mtu]]
mtu: int::
Interface MTU (0 if unknown)

[[discoveredinterface-name]]
name: <<class-string,String>>::
Interface name

[[discoveredinterface-pic]]
pic: int::
Phisical location

[[discoveredinterface-port]]
port: int::
Port number

[[discoveredinterface-speed]]
speed: int64::
Speed of the interface

[[discoveredinterface-type]]
type: int::
<<enum-interface-types,Interface type>>
--
