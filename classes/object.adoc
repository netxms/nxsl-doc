[[class-object]]
=== Object

Class Object is the root of the class hierarchy. Every class has Object as a superclass.

==== Instance attributes

[.attr-list]
--
[[object-__class]]
__class => Class::
Object of <<class-class>> class
--

==== Instance methods

[.method-list]
--
