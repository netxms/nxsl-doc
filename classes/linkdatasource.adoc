[.nxsl-class]
[[class-linkdatasource]]
=== LinkDataSource

Data source item for Network Map link

==== Instance attributes

[.attr-list]
--
[[linkdatasource-dci]]
dci => <<class-dci>>::
Data source DCI item

[[linkdatasource-dciId]]
dciId: int::
Data source DCI item id

[[linkdatasource-format]]
format: <<class-string,String>>::
Data display format

[[linkdatasource-location]]
location: int::
Location of the link data source. Possible values: 0 = CENTER, 1 = OBJECT1, 2 = OBJECT2.

[[linkdatasource-object]]
object => <<class-datacollectiontarget>>::
DCI source object

[[linkdatasource-objectId]]
objectId: int::
DCI source object id
--
