[.nxsl-class]
[[class-webserviceresponse]]
=== WebServiceResponse

Contains all information about web service custom request execution result

==== Instance attributes

[.attr-list]
--
[[webserviceresponse-agentErrorCode]]
agentErrorCode: int::
Agent error code

[[webserviceresponse-body]]
body: <<class-string,String>>::
Response body. Alias for `document`.

[[webserviceresponse-document]]
document: <<class-string,String>>::
Document that was returned by web service as a response to the request

[[webserviceresponse-errorMessage]]
errorMessage: <<class-string,String>>::
Human readable error message set by server or agent

[[webserviceresponse-httpResponseCode]]
httpResponseCode: int::
HTTP response code

[[webserviceresponse-statusCode]]
statusCode: int::
HTTP response status code. Alias for `httpResponseCode`.

[[webserviceresponse-success]]
success: bool::
Will return `True` in case of success and `False` if there was problem on server or on agent
--
