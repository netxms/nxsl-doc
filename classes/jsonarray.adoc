[.nxsl-class]
[[class-jsonarray]]
=== JsonArray

Represents JSON array object.

[NOTE]
To parse string into JsonObject use <<func-jsonparse>>.

==== Instance attributes

[.attr-list]
--
[[jsonarray-size]]
size: int::
Number of elements in array

[[jsonarray-values]]
values: <<class-array,Array>>::
NXSL array with JSON array elements
--
