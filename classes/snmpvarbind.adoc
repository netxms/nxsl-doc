[.nxsl-class]
[[class-snmpvarbind]]
=== SNMPVarBind

Represents an SNMP varbind concept in NetXMS. A varbind logically consists of an OID and a value.

==== Instance attributes

[.attr-list]
--
[[snmpvarbind-name]]
name: <<class-string,String>>::
Object name (OID string).

[[snmpvarbind-type]]
type: int::
ASN.1 type.

[[snmpvarbind-value]]
value: <<class-string,String>>::
Object value as a string.

[[snmpvarbind-printableValue]]
printableValue: <<class-string,String>>::
Object value as a string with non-printable characters replaced by `?` character.

[[snmpvarbind-valueAsIp]]
valueAsIp: <<class-string,String>>::
Object value IP address, represented as string.

[[snmpvarbind-valueAsMac]]
valueAsMac: <<class-string,String>>::
Object value as MAC address, represented as string.
--

==== Methods

[.method-list]
--
[[snmpvarbind-getValueAsByteStream]]
getValueAsByteStream() => ByteStream::
Returns object value as <<class-bytestream>>.

[[snmpvarbind-getValueAsString]]
getValueAsString(codepage): <<class-string,String>>::
Constructs a new String by decoding the varbind value using the specified codepage.
--
