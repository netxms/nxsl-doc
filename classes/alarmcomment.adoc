=== AlarmComment

Represents NetXMS alarm comment.

==== Instance attributes

`id => Integer`::
Alarm comment ID.

`changeTime => Integer`::
link:https://en.wikipedia.org/wiki/Unix_time[Unix timestamp] of the alarm comment last modification time.

`userId => Integer`::
ID of user who last modified this alarm commant.

`text => String`::
Alarm comment text.
