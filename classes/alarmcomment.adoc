[[class-alarmcomment]]
=== AlarmComment

Represents NetXMS alarm comment.

==== Instance attributes

[.attr-list]
--
[[alarmcomment-id]]
id: int::
Alarm comment ID.

[[alarmcomment-changeTime]]
changeTime: int::
link:https://en.wikipedia.org/wiki/Unix_time[Unix timestamp] of the alarm comment last modification time.

[[alarmcomment-userId]]
userId: int::
ID of user who last modified this alarm comment.

[[alarmcomment-user]]
user => <<class-userdbobject>>::
User who last modified this alarm comment (class <<class-userdbobject>>).

[[alarmcomment-text]]
text: <<class-string,String>>::
Alarm comment text.
--
