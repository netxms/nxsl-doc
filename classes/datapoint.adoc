[.nxsl-class]
[[class-datapoint]]
=== DataPoint

Represents a single data point containing a timestamp and value pair. DataPoint objects are typically returned by functions that retrieve historical DCI values, such as <<func-getdcivaluesex,GetDCIValuesEx()>>.

==== Instance attributes

[.attr-list]
--
[[datapoint-timestamp]]
timestamp: int64::
Timestamp of the data point in milliseconds since epoch (1 Jan 1970 00:00:00 UTC).
+
[.version-changed]
Changed in version 6.0: timestamp is now in milliseconds instead of seconds.

[[datapoint-value]]
value: <<class-string,String>>::
The value associated with the timestamp.
--
