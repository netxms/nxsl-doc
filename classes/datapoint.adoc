[.nxsl-class]
[[class-datapoint]]
=== DataPoint

Represents a single data point containing a timestamp and value pair. DataPoint objects are typically returned by functions that retrieve historical DCI values, such as <<func-getdcivaluesex,GetDCIValuesEx()>>.

==== Instance attributes

[.attr-list]
--
[[datapoint-timestamp]]
timestamp: int::
link:https://en.wikipedia.org/wiki/Unix_time[Unix timestamp] of the data point.

[[datapoint-value]]
value: <<class-string,String>>::
The value associated with the timestamp.
--
