[.nxsl-class]
[[class-softwarepackage]]
=== SoftwarePackage

Represents software package installed on <<class-node>> object.

==== Instance attributes

[.attr-list]
--
[[softwarepackage-changeCode]]
changeCode: int::
Change code one of next:
+
* CHANGE_NONE: 0
* CHANGE_ADDED: 1
* CHANGE_UPDATED: 2

[[softwarepackage-date]]
date: <<class-string,String>>::
Change date in YYYYMMDD format

[[softwarepackage-description]]
description: <<class-string,String>>::
PAckage description

[[softwarepackage-name]]
name: <<class-string,String>>::
Package name

[[softwarepackage-timestamp]]
timestamp: int::
Package change date as UNIX timestamp

[[softwarepackage-uninstallKey]]
uninstallKey: <<class-string,String>>::
Get product uninstall key if available.

[[softwarepackage-url]]
url: <<class-string,String>>::
PAckage URL

[[softwarepackage-vendor]]
vendor: <<class-string,String>>::
PAckage vendor

[[softwarepackage-version]]
version: <<class-string,String>>::
Package version
--
