[.nxsl-class]
[[class-user]]
=== User

Object represent user object and extends <<class-userdbobject>>.

==== Instance attributes

[.attr-list]
--
[[user-authMethod]]
authMethod: int::
<<enum-auth-methods,Authentication method>>.

[[user-certMappingData]]
certMappingData: <<class-string,String>>::
Data that will be used to mpa certificate

[[user-certMappingMethod]]
certMappingMethod: int::
<<enum-mapping-methods,Certificate mapping methods>>.

[[user-disabledUntil]]
disabledUntil: <<class-string,String>>::
UNIX timestamp with date until this user is disabled

[[user-email]]
email: <<class-string,String>>::
Email set in user properties

[[user-fullName]]
fullName: <<class-string,String>>::
User full name

[[user-graceLogins]]
graceLogins: int::
Grace login count

[[user-phoneNumber]]
phoneNumber: <<class-string,String>>::
Phone number set in user properties

[[user-lastLogin]]
lastLogin: int::
UNIX timestamp with last user's login time
--
