[.nxsl-class]
[[class-component]]
=== Component

 Node components

==== Instance attributes

[.attr-list]
--
[[component-class]]
class: <<class-string,String>>::
Type of the component:
+
* `unknown`
* `chassis`
* `backplane`
* `container`
* `power supply`
* `fan`
* `sensor`
* `module`
* `port`
* `stack`

[[component-children]]
children: <<class-array,Array>>::
List of direct children (Array of <<class-component>> object intances).

[[component-description]]
description: <<class-string,String>>::
Component description

[[component-firmware]]
firmware: <<class-string,String>>::
Component firmware version, if available.

[[component-ifIndex]]
ifIndex: int::
Interface index number

[[component-model]]
model: <<class-string,String>>::
Component model number, if available.

[[component-name]]
name: <<class-string,String>>::
Component name, if available.

[[component-serial]]
serial: <<class-string,String>>::
Component serial number, if available.

[[component-vendor]]
vendor: <<class-string,String>>::
Component vendor, if available.
--
