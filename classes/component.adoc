[.nxsl-class]
[[class-component]]
=== Component

 Node components

==== Instance attributes

[.attr-list]
--
[[component-class]]
class => String::
Type of the component:
+
* `unknown`
* `chassis`
* `backplane`
* `container`
* `power supply`
* `fan`
* `sensor`
* `module`
* `port`
* `stack`

[[component-children]]
children => Array::
List of direct children (Array of <<class-component>> object intances).

[[component-description]]
description => String::
Component description

[[component-firmware]]
firmware => String::
Component firmware version, if available.

[[component-ifIndex]]
ifIndex => Integer::
Interface index number

[[component-model]]
model => String::
Component model number, if available.

[[component-name]]
name => String::
Component name, if available.

[[component-serial]]
serial => String::
Component serial number, if available.

[[component-vendor]]
vendor => String::
Component vendor, if available.
--
