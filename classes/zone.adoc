[[class-zone]]
=== Zone

Represents network zone. Extends <<class-netobj>>.

==== Instance attributes

[.attr-list]
--
include::_shared/netobj-attrs.adoc[]

[[zone-proxyNodes]]
proxyNodes: <<class-array,Array>><Node>::
Array of <<class-node>> objects that are currently set as proxies for this zone.

[[zone-proxyNodeIds]]
proxyNodeIds: <<class-array,Array>><Integer>::
Array of integers representing identifiers of node objects that are currently set as proxies for this zone.

[[zone-uin]]
uin: int::
Zone UIN (Unique Identification Number).
