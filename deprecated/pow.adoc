[[func-pow]]
== pow()

****
[IMPORTANT]
====
*Deprecated in 5.0.*

Please use <<func-math-pow>> instead.
====
****

[source,c]
----
pow(x, y) => Float
----

Calculates x raised to the power of y.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Float|Initial value.
|y|Float|Power.
|===

.Return
x raised to the power of y.

[.source]
....
println(pow(2, 3)); //Will print "8"
....
