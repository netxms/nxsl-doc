== tan()

****
[IMPORTANT]
====
*Deprecated in 5.0.*

Please use <<func-math-tan>> instead.
====
****

[source,c]
----

----

Calculates tangent x

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Float|Angle in radian
|===

.Return
Result of tangent for this angle

[source,c]
----
