[.nxsl-function]
[[func-sin]]
== sin()

****
[IMPORTANT]
====
*Deprecated in 5.0.*

Please use <<func-math-sin>> instead.
====
****

[source,c]
----
sin(x): float
----

Calculates sine from given angle in radians.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Float|Angle in radian
|===

.Return
Result of sine for this angle

[source,c]
----
print(sin(0.5)); //will print 0.479426
----
