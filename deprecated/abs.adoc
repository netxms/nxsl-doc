[[func-abs]]
== abs()

****
[IMPORTANT]
====
*Deprecated in 5.0.*

Please use <<func-math-abs>> instead.
====
****

`abs(number) => Float`::

Returns the absolute value of the number.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|number|Float|Input value.
|===

.Return

Absolute value of the input.

[.source]
....
>>> abs(12.3)
12.3
>>> abs(-0.307)
0.307
....
