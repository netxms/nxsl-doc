[[func-max]]
== max()

****
[IMPORTANT]
====
*Deprecated in 5.0.*

Please use <<func-math-max>> instead.
====
****

[source,c]
----
max(value1, value2, ...): void
----

Returns maximal value from a list of values.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|values|Number|Values separated by comma
|===

.Return
Maximal value.

[.source]
....
max(2, 3, 4, 8); //Will print "8"
....
