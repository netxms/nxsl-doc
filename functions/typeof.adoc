[[func-typeof]]
== typeof()

[source,c]
----
typeof(instance) => string
----

Return type of the `instance`.

.Parameters
[cols="1,3" grid="none", frame="none"]
|===
|instance|Instance of the object or primitive.
|===

.Return

Name of the type.

.Example
[.output]
....
>>> typeof(1)
int32
>>> typeof(1L)
int64
>>> typeof(%(1, 2, 3))
array
>>> typeof(new Table())
object
>>> typeof(null)
null
....
