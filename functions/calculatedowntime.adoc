[.nxsl-function]
[[func-calculatedowntime]]
== CalculateDowntime()

[source,c]
----
CalculateDowntime(object, schedule, startTime, endTime) => Array
----

Calculates downtime periods for a given network object based on a maintenance schedule within the specified time range.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|object|<<class-netobj>> or Integer|Network object or object ID
|schedule|String|Maintenance schedule name
|startTime|Integer|Start of the period (as UNIX timestamp)
|endTime|Integer|End of the period (as UNIX timestamp)
|===

.Return

Array of <<class-downtimeinfo,DowntimeInfo>> objects describing downtime windows, or NULL on failure.

.Example
[.source]
....
// Calculate downtime for the last 24 hours
downtimes = CalculateDowntime($node, "business-hours", time() - 86400, time());
if (downtimes != null) {
    foreach (d : downtimes) {
        println("Downtime from " .. d->startTime .. " to " .. d->endTime);
    }
}
....
