[[func-log10]]
== log10()

[source,c]
----
log10(x) => Number
----

Calculates logarithm of given value to base 10.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Number|Number to calculate logarithm of
|===

.Return
Logarithm of x to base 10.

.Example
[.source]
....
pritnl log10(2); // Will print "0.301030"
....
