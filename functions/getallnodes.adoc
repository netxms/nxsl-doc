[[func-getallnodes]]
== GetAllNodes()

GetAllNodes() => array::
Get list of all <<class-node>> objects in the system as array.
+
.Return
Array of node objects.
+
[.source]
....
>>> for (n : GetAllNodes()) {
>>>   println(n.id . " - " . n.name);
>>> }

6766 - demo-netxms
6901 - Control Unit 1
6902 - Control Unit 2
....
