[[func-pow]]
== pow()

[source,c]
----
pow(x, y) => Number
----

Calculates x raised to the power of y.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Number|Initial value.
|y|Number|Power.
|===

.Return
x raised to the power of y.

.Example
[.source]
....
println(pow(2, 3)); //Will print "8"
....
