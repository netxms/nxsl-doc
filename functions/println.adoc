[.nxsl-function]
[[func-println]]
== println()

println(object) => void::
Prints any text or text representation of object and adds new line symbol after it
+
.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|object|object|Any object 
|===
+
[source,c]
----
a = ["a", 1, 1.5];
println("Array: ");
println(a); 
//Array: 
//[a, 1, 1.500000]
----
