[[func-io-deletefile]]
== IO::DeleteFile()

[source,c]
----
IO::DeleteFile(file) => Boolean
----

Deletes file.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|file|String|File to delete
|===

.Return
`TRUE` if delete was successful and `FALSE` if not.

[.source]
....
IO::DeleteFile("/tmp/file") // Will delete "/tmp/file" file.
....
