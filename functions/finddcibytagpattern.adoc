[[func-finddcibytagpattern]]
== FindDCIByTagPattern()

[source,c]
----
FindDCIByTagPattern(node, dciUserTagPattern) => Integer
----

Find ID of the DCI on node by tag using pattern matching. 
Glob symbols `?` and `*` can be used. 
Function returns only one match. <<func-findalldcis>> can be used to get
multiple matches. *This function was added in version 5.2*

.Parameters
[cols="1,3" grid="none", frame="none"]
|===
|node|<<class-node>> object instance (e.g. `$node`)
|dciUserTagPattern|DCI user tag pattern
|===

.Return

Integer ID of the DCI or 0 if not found.
