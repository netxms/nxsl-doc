[.nxsl-function]
[[func-print]]
== print()

print(object) => void::
Prints any text or text representation of object 
+
.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|object|object|Any object 
|===
+
[source,c]
----
a = ["a", 1, 1.5];
print("Array: ");
print(a); // Will print "Array: [a, 1, 1.500000]"
----
