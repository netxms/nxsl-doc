== Math::Sqrt()

[source,c]
----
Math::Sqrt(input) => Float
----

Gets square root from number.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|input|Float|Input number.
|===

.Return
Square root value.

.Example
[.source]
....
>>> Math::Sqrt(4.5)
2.121320
....
