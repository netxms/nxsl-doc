== Math::Pow

Math::Pow(x, y): number::
Calculates x raised to the power of y.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|number|Base
|y|number|Power
|===

.Return
x raised to the power of y.

[source,c]
----
println(Math::Pow(2.5, 3.1)); // Will print "17.124347"
----
