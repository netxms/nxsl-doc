[[func-math-tanh]]
== Math::Tanh()

[source,c]
----
Math::Tanh() => void
----

Calculates hyperbolic tangent x

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Number|Angle in radian 
|===

.Return
Result of hyperbolic tangent for this angle

.Example
[source,c]
----
print(Math::Tanh(0.5)); //will print 0.462117
----
