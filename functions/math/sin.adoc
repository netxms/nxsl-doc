[[func-math-sin]]
== Math::Sin()

[source,c]
----
Math::Sin(x) => Float
----

Calculates sine from given angle in radians. 

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Number|Angle in radians
|===

.Return
Result of sine for this angle

.Example
[source,c]
----
print(Math::Sin(0.5)); //will print "0.479426"
----
