[[func-math-abs]]
== Math::Abs()

` Math::Abs(number) => Float`::

Returns the absolute value of the number.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|number|Float|Input value.
|===

.Return

Absolute value of the input.

.Example
[.source]
....
>>> Math::Abs(12.3)
12.3
>>> Math::Abs(-0.307)
0.307
....
