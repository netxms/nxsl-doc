[[func-math-cos]]
== Math::Cos()

[source,c]
----
Math::Cos(x) => Float
----

Calculates cosine from given angle in radians. 

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Float|Angle in rad
|===

.Return
Result of cosine for this angle

.Example
[source,c]
----
print(Math::Cos(0.5)); //will print 0.877583
----
