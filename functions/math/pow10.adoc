== Math::Pow10()

[source,c]
----
Math::Pow(x) => Number
----

Calculates 10 raised to the power of x.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Integer|Power
|===

.Return
x raised to the power of 10.

.Example
[source,c]
----
println(Math::Pow10(2)); // Will print "100.000000"
----
