== Math::Pow10()

[source,c]
----
Math::Pow(x) => Float
----

Calculates x raised to the power of 10.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|Float|Initial value.
|===

.Return
x raised to the power of 10.

.Example
[.source]
....
println(Math::Pow10(2)); //Will print "100.000000"
....
