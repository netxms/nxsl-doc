== Math::Sinh

Math::Sinh(x): float::
Calculates hyperbolic sine x.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|x|number|Angle in radians 
|===

.Return
Result of hyperbolic sine for this angle

[source,c]
----
print(Math::Sinh(0.5)); // Will print "0.521095"
----
