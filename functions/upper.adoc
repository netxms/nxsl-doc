[.nxsl-function]
[[func-upper]]
== upper()

[source,c]
----
upper(string) => String
----

Converts string to uppercase.

.Parameters
[cols="1,1,3" grid="none", frame="none"]
|===
|string|String|String to convert
|===

.Return
Source string converted to uppercase.

.Example
[.source]
....
print upper("aBcD"); //will print "ABCD"
....
